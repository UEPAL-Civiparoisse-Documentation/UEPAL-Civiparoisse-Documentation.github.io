# 06 - Gestion de groupes

## 001 - Créer un groupe normal

1. Allez sur le menu Contacts et choisissez le menu Gérer les groupes.
2. Cliquez sur le bouton Ajouter un groupe.
3. Renseignez les champs Titre du groupe public, et Titre du groupe.
4. Choisissez comme Type de groupe : « Liste de diffusion ».
5. Ne changez pas la visibilité.
6. Cliquez sur Enregistrer.
7. CONFIRMEZ que vous arrivez sur la page « Ajouter au groupe… »
8. Saisissez le nom ou adresse mail d’une personne qui est déjà dans la base.
9. Cochez la case blanche à gauche du nom de la personne. Si plusieurs noms sont affichés, vous avez la possibilité de cocher plusieurs cases.
10. Cliquez sur le bouton « Ajouter les contacts au Groupe… ».
11. Validez en cliquant sur le bouton « Ajouter au groupe ».
12. Allez dans le menu Rechercher dans le bandeau du haut, et cliquez sur Rechercher des contacts.
13. Sur la ligne « Dans », choisissez le groupe créé ci-dessus, et cliquez sur Rechercher.
14. CONFIRMEZ que la/les personnes que vous avez rajoutés dans le groupe sont bien affichés dans la liste.

## 002 - Créer un groupe dynamique

1. Allez dans le menu Rechercher, et cliquez sur Recherche Avancée.
2. Sur la page de recherches, saisissez un ou plusieurs critères (par exemple, code postal = 67000 ; ou civilité = Mme).
3. Cliquez sur Rechercher.
4. Si aucune liste ne s’affiche, changez les critères ci-dessus.
5. En haut de la liste, sur la ligne Sélectionner, choisissez « Tous ».
6. Juste en dessous, dans les Actions, choisissez « Groupe – créer un groupe dynamique ».
7. Donnez un titre à ce groupe dynamique
8. Cochez également la case Liste de diffusion, et cliquez sur Enregistrer.
9. CONFIRMEZ que le bouton Terminé s’affiche.
10. Vous venez de créer un groupe qui liste toutes les fiches répondant aux critères saisis ci-dessus (exemple tous les codes postaux = 67000).
11. Allez sur une fiche qui ne vérifie pas les critères saisis ci-dessus (exemple : dont le code postal n’est pas 67000).
12. Modifiez la fiche pour lui rajouter les critères saisis ci-dessus (exemple : en modifiant le code postal à 67000)
13. Allez dans le menu Rechercher dans le bandeau du haut, et cliquez sur Rechercher des contacts.
14. Sur la ligne « Dans », choisissez le groupe créé ci-dessus, et cliquez sur Rechercher (le groupe devrait s’afficher avec une étoile * devant son nom, pour indiquer qu’il s’agit d’un groupe dynamique).
15. CONFIRMEZ que la personne que vous avez modifié les critères est bien affiché dans la liste.

## 003 - Faire une recherche de toutes les personnes appartenant à un groupe donné

1. Allez dans le menu Rechercher / Rechercher des contacts.
2. Sur la ligne « Dans », choisissez un des groupes existants, et cliquez sur « Rechercher ».
3. CONFIRMEZ qu’une liste des personnes (et/ou de foyers) s’affiche.
4. Sélectionnez une des fiches, en cliquant sur la case blanche à sa gauche.
5. Allez dans le menu Actions, et choisissez dans la liste déroulante le choix « Groupe – retirer le contact ».
6. Sélectionnez le nom du groupe dans le menu déroulant.
7. Validez que vous voulez supprimer du groupe.
8. CONFIRMEZ que la liste qui s’affiche ne contient plus la fiche en question.
9. Recherchez la fiche via les outils de recherche
10. Cliquez sur l’onglet Groupes
11. CONFIRMEZ que le nom du groupe n’apparait pas.

## 004 - Affichage des Groupes sur la fiche d’un Individu

1. Cliquez sur l’onglet « Groupe » dans une fiche individu. 
2. CONFIRMEZ que la liste des groupes auxquels il appartient s’affiche correctement.
