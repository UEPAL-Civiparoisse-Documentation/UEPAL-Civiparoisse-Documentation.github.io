# 11 - Edition de listes et d’étiquettes (autocollantes)

## 001 - Test du bon fonctionnement des listes

1. Cliquez sur l’icône « Listes » du menu Civi Paroisse.
2. Cliquez sur chaque liste.
3. CONFIRMEZ que chaque liste s’ouvre dans un nouvel onglet, et que des résultats s’affichent sur la liste.

Listes à tester :

    a. Dates d’anniversaires
    b. Nouveaux arrivants
    c. Foyers Paroissiens
    d. Naissances
    e. Baptêmes
    f. Présentations
    g. Confirmations
    h. Bénédictions Nuptiales
    i. Décès
    j. Liste Electorale
    k. Conseil Presbytéral
    l. Foyers sans Adresses Mails
    m. Liste des Foyers par Quartiers
    n. Liste des Quartiers

## 002 - Consulter et imprimer les listes "groupes"

1. Cliquez sur l’icône « Listes » du menu Civi Paroisse.
2. Dans la partie Groupes / Participants, choisissez la liste Dates d’anniversaires.
3. Dans les critères de choix, sélectionnez un groupe.
4. Dans la liste, sélectionnez 5 à 10 personnes, puis cliquez sur « Actions » et « Imprimer / fusionner un document ».
5. CONFIRMEZ qu’une fenêtre s’ouvre.
6. Renseignez un « sujet de l’activité ».
7. Sélectionnez le champ de fusion « nom de famille », et ajoutez éventuellement d’autres champs de fusion.
8. Saisissez un texte dans « corps du document ».
9. Cliquez sur « prévisualisation », un téléchargement au format PDF se fait.
10. Ouvrez le fichier téléchargé.
11. CONFIRMEZ que le document est cohérent avec les champs de fusion et le texte saisis ci-dessus.

## 003 - Consulter et imprimer les listes "membres"

5. Renseignez un « sujet de l’activité ».
6. Sélectionnez le champ de fusion « nom de famille », et ajoutez éventuellement d’autres champs de fusion.
7. Saisissez un texte dans « corps du document ».
8. Cliquez sur « prévisualisation », un téléchargement au format PDF se fait.
9. Ouvrez le fichier téléchargé.
10. CONFIRMEZ que le document est cohérent avec les champs de fusion et le texte saisis ci-dessus.

## 004 - Exporter au format Excel une sélection

1. Cliquez sur l’icône « Listes » du menu Civi Paroisse.
2. Dans la partie Paroisse / Elections, choisissez la liste électorale.
3. Sous le bouton Action, sélectionnez quelques fiches en cliquant sur les cases à gauche de la ligne.
4. Cliquer sur le bouton Action.
5. Sélectionner Download Spreadsheat.
6. Dans la fenêtre qui s’ouvre, choisissez le format souhaité (Excel).
7. Cliquer sur Télécharger.
8. CONFIRMEZ qu’un fichier Excel a bien été téléchargé.
9. Ouvrez le fichier Excel.
10. CONFIRMEZ que le fichier contient bien les fiches sélectionnées, ainsi que les informations affichées sur la liste.

## 005 - Editer la liste électorale

1. Créer un nouvel Individu, en utilisant le bouton sur la page d'accueil.
2. Dans le lien avec la Paroisse, indiquez qu'il est électeur.
3. A partir des listes disponibles en page d'accueil, allez sur la liste électorale.
4. CONFIRMEZ que vous retrouvez bien le nouvel Individu dans la liste électorale.
5. Refaire le même exercice de création d'un nouvel Individu, mais en indiquant un autre lien avec la paroisse.
6. CONFIRMEZ que ce deuxième nouvel individu n'apparait pas sur la liste électorale.

## 006 - Liste des nouveaux arrivants

1. Créer un nouvel Individu, en utilisant le bouton sur la page d'accueil.
2. A partir des listes disponibles en page d'accueil, allez sur la liste des Nouveaux Arrivants.
3. CONFIRMEZ que vous retrouvez bien le nouvel Individu dans la liste.

## 007 - Liste des Foyers sans adresses mails

1. Créer un nouvel Foyer en utilisant le bouton sur la page d'accueil.
2. Dans la foulée, créer un nouvel Individu en utilisant le formulaire disponible après la création du foyer.
3. N’indiquer pas d’adresses mail pour ce nouvel Individu.
4. Allez sur la page d’accueil, et cliquer sur les Listes
5. Choisir la liste « Foyers sans adresses mails »
6. CONFIRMEZ que le Foyer nouvellement crée est bien affiché dans la liste.
7. Retourner sur la fiche Individu que vous venez de créer.
8. Rajouter une adresse mail sur la fiche.
9. Retourner sur la liste « Foyers sans adresses mails ».
10. CONFIRMEZ que le Foyer n’est plus affiché dans la liste.

## 008 - Créer des étiquettes autocollantes pour tous les nouveaux arrivants

1. A partir des listes disponibles en page d’accueil, allez sur la liste des Nouveaux Arrivants.
2. Sélectionner l’ensemble des lignes, en cliquant sur le premier carré de sélection.
3. Cliquer sur le bouton Action.
4. Choisir le menu Etiquettes pour la Poste – Imprimer.
5. Dans la fenêtre qui s’ouvre, sélectionner un type d’étiquette.
6. Renseigner les autres paramètres comme souhaitée.
7. Cliquer sur le bouton Créer les étiquettes de publipostage.
8. CONFIRMEZ l’ouverture d’un fichier PDF avec les noms et adresses.

## 009 - Créer un groupe avec tous les Amis de la paroisse

1. Dans le menu Rechercher, cliquer sur Rechercher avancée.
2. Dans l’entrée Adhésions, choisir le champ Type de Membre.
3. Choisir Ami de la paroisse.
4. Cliquer sur Rechercher.
5. CONFIRMEZ qu’une liste s’affiche.
6. Cliquer sur Sélectionner Tous au-dessus de la liste.
7. Dans le menu Actions, sélectionner Groupes – ajouter les contacts.
8. Si le groupe existe déjà, choisir Ajouter le contact à un groupe existant, et sélectionner le nom du groupe dans le menu déroulant.
9. Si le groupe n’existe pas, choisir Créer un nouveau groupe, et indiquer le nom du groupe.
10. Cliquer sur Ajouter au Groupe.
11. Dans le menu Rechercher, cliquer sur Rechercher des contacts.
12. Dans le champ Groupe, indiquer le nom du Groupe choisi ci-dessus.
13. CONFIRMEZ que les Amis de la paroisse sont bien présents dans le Groupe.

## 010 - Exporter dans Excel tous les nouveaux arrivants

1. Sur la page d’accueil, cliquez sur le bouton Listes.
2. Choisissez la liste Nouveaux arrivants.
3. Modifiez le choix des critères si vous le souhaitez.
4. CONFIRMEZ que la liste affiche des informations.
5. Sélectionnez toutes les fiches, en cliquant sur la première case sous le bouton Action.
6. CONFIRMEZ que toutes les fiches sont sélectionnées.
7. Cliquez sur le bouton Action.
8. Cliquer sur Download Spreadsheet (« télécharger un fichier Excel »).
9. Choisissez le format Excel (ou Open Office, selon le logiciel installé sur votre ordinateur).
10. Cliquez sur Télécharger.
11. CONFIRMEZ qu’un fichier est téléchargé.
12. Ouvrez le fichier.
13. CONFIRMEZ que le contenu du fichier est le même que celle de la liste affichée à l’écran.
