# 08 - Gestion des étiquettes/tag

## 001 - Créer une étiquette/tag

1. Dans le menu « contacts », sélectionnez « gérer les tags/étiquettes ».
2. Cliquez sur « ajouter un tag/étiquette ».
3. Remplissez les champs demandés dans la fenêtre qui s’ouvre.
4. Ouvrez la fiche d’un contact.
5. Sur la fiche de synthèse, cliquez sur le lien Tags/Etiquettes.
6. Choisissez l’étiquette créé ci-dessus.
7. CONFIRMEZ la bonne prise en compte des données (dont éventuellement un tag/étiquette parent et/ou une couleur).

## 002 - Appliquer une étiquette/tag à un ensemble d'individus

1. Lancez une recherche de contacts uniquement sur des Individus particuliers.
2. Sélectionnez une lettre initiale, puis sélectionnez/cochez 5 contacts (carrés à gauche) et notez les 5 noms sélectionnés.
3. Dans le menu déroulant « Actions » sélectionnez « Tag/Etiquette – Ajouter aux contacts ».
4. Sélectionnez une étiquette.
5. Valider en cliquant sur « taguer les contacts ».
6. Cliquer sur « modifier les critères de recherche ».
7. Sélectionner le nom du tag/étiquette dans la ligne « Avec ».
8. CONFIRMEZ que les 5 contacts apparaissent bien dans la liste.

## 003 - Appliquer une étiquette/tag à un ensemble d'individus, de foyers et d'organisations

1. Lancez une recherche de contacts sans préciser le type de contacts.
2. Sélectionnez une lettre initiale puis des contacts avec au minimum 1 individu et 1 foyer (voir icône à gauche comme repère du type de contact) et notez les noms sélectionnés.
3. Dans le menu déroulant « Actions », sélectionnez « Tag/Etiquette – Ajouter aux contacts ».
4. Sélectionnez une étiquette.
5. Validez en cliquant sur « Taguer les contacts ».
6. Cliquer sur « Modifier les critères de recherche ».
7. Sélectionner le nom du tag/étiquette dans la ligne « Avec ».
8. CONFIRMEZ que les contacts sélectionnés apparaissent bien sur la liste.

## 004 - Faire une recherche de toutes les personnes ayant une certaine étiquette

1. Allez dans le menu Rechercher.
2. Dans le menu Avec tout tag/étiquette, choisissez une des étiquettes proposées.
3. CONFIRMEZ qu’une liste d’Individus et de Foyers s’affichent.
4. Si aucune liste ne s’affiche, testez avec l’étiquette Testeurs.
5. CONFIRMEZ qu’une liste de contacts s’affichent.

## 005 - Affichage des étiquettes / tags sur la fiche d’un Individu

1. Allez dans le menu Rechercher.
2. Dans le menu Avec tout tag/étiquette, choisissez une des étiquettes proposées.
3. Ouvrez une des fiches ainsi affichées dans la liste.
4. CONFIRMEZ que le Tag/étiquette sélectionné dans le point 2 s’affiche sur la fiche à l’écran.
