# 10 - Recherches et actions

## 001 - Rechercher un foyer

1. En utilisant les outils de recherche, recherchez un Foyer.
2. Au moins deux possibilités existent pour effectuer une recherche Foyer. Tester chaque possibilité et CONFIRMEZ que vous trouvez une fiche Foyer.
3. Dans la barre de menu en haut, cliquer sur le triangle vert (logo de CiviCRM).
4. Saisissez Rechercher dans la case blanche qui apparait.
5. Utilisez les différentes fonctionnalités de recherche pour rechercher un Foyer (attention, certaines recherches ne permettent pas de rechercher un Foyer. Nous vous laissons découvrir lesquelles !).

## 004 - Afficher toutes les personnes d’un foyer

Plusieurs méthodes sont possibles (essayez d'effectuer tous les tests).  
***Méthode 1 : si l’ensemble du foyer porte le même nom de famille***

1. Dans le moteur de recherche en haut à gauche de l’écran, saisir le nom du foyer, et uniquement le nom du foyer.
2. Taper sur la touche Entrée du clavier.
3. CONFIRMEZ que s’affiche alors l’ensemble des Individus composant le Foyer, ainsi que le Foyer lui-même.

***Méthode 2 : si l’ensemble du foyer ne porte pas le même nom de famille, ou si vous avez un doute***

1. Ouvrez la Recherche Avancée dans le menu Rechercher.
2. Dans le champ Type des résultats, cliquer sur « Contacts liés ».
3. Dans le champ Type de relation, choisissez « Membre du foyer de ».
4. Saisissez le nom de famille d’un des membres du foyer (en rajouter le signe % de part et d’autre).
5. CONFIRMEZ que l’ensemble des membres du foyer s’affiche.

## 005 - Afficher tous les Individus présents dans la base

1. Dans le menu Rechercher, cliquer sur Rechercher des contacts.
2. Dans la zone de Sélection Contact, sélectionner Particulier.
3. CONFIRMEZ que l’ensemble des Individus de la base s’affichent (vous devriez en trouver autour de 1500-1550 sur test-1).

## 006 - Afficher toutes les personnes d’un même genre

1. Ouvrez la Recherche Avancée dans le menu Rechercher.
2. Dans la partie Données démographiques, choisissez le genre souhaité.
3. CONFIRMEZ que les fiches qui s’affichent sont bien des fiches du genre recherchée.

## 007 - Afficher les personnes ayant entre 25 et 45 ans au 5 janvier dernier

1. Ouvrez la Recherche Avancée dans le menu Rechercher.
2. Ouvrez la partie Données démographiques.
3. Saisissez l’âge minimal (25) et l’âge maximal (45).
4. Dans le champ Au, indiquer la date du 5 janvier.
5. Vérifiez quelques fiches, pour valider que l’âge est correct.
6. CONFIRMEZ que vous trouvez des fiches Individus (environ 255 fiches sur test-1).

## 008 - Rechercher les foyers où nous n’avons pas d’adresses-mails

1. A partir de la page d’accueil, cliquer sur l’icône Liste
2. Cliquer sur la liste Foyers sans adresses mails.
3. A partir de la liste qui s’affiche, vérifier, pour quelques résultats affichés, que les Chefs de famille composant le Foyer n’ont pas d’adresse mail.
4. CONFIRMEZ que des fiches Foyers s’affichent (environ 600 – 610 fiches sur test-1).
5. Ouvrez la fiche Individu d’un des Chefs de famille ainsi ciblés.
6. Renseignez son adresse mail.
7. Relancez la recherche en effectuant les points 1 et 2 ci-dessus.
8. CONFIRMEZ que le Foyer n’apparait plus dans la recherche.

## 009 - Afficher les parents d’enfants du KT

1. Dans le menu Recherche, ouvrir la Recherche Avancée.
2. Ouvrer l’entrée Paramètres de recherche.
3. Dans le menu Type des résultats, sélectionner le menu Contacts liés.
4. Sélectionner le type de relation Parent de.
5. Dans l’entrée Critères de base, choisir le Groupe Groupe enfant KT.
6. Cliquer sur Rechercher.
7. CONFIRMEZ qu’une liste d’individu s’affiche.
8. Ouvrir quelques fiches Individus.
9. Sur chaque fiche, CONFIRMEZ la présence d’une relation Parent de.
10. CONFIRMEZ, en cliquant sur la fiche de l’enfant, que celui-ci est bien noté dans le groupe Groupe Enfant KT.

## 010 - Rechercher toutes les compétences Chant

1. Dans le menu Recherche, ouvrir la Recherche Avancée.
2. Dans l’entrée Champs personnalisées, Compétences, sélectionner une des compétences Musique :chant.
3. Rajouter les trois autres compétences Chant.
4. Cliquer sur Rechercher.
5. CONFIRMEZ que des fiches s’affichent (au moins une petite quinzaine sur test-1).
6. Ouvrir quelques fiches.
7. CONFIRMEZ qu’une des compétences Chant est bien visible dans chaque fiche.

## 011 - Afficher toutes les Individus qui ont une Note dans leur fiche

1. Dans le menu Recherche, ouvrir la Recherche Avancée.
2. Ouvrir l’entrée Notes.
3. Dans le champ Texte présent dans la note, saisissez %.
4. Et à côté, sélectionner Les deux.
5. Cliquer sur Rechercher.
6. CONFIRMEZ qu’une liste de fiches s’affiche (environ 250 fiches sur test-1).
7. Ouvrir quelques fiches.
8. CONFIRMEZ qu’une note est bien présente pour chaque fiche ouverte.

## 012 - Créer des étiquettes autocollantes pour tous les trentenaires

1. Dans le menu Recherche, ouvrir la Recherche Avancée.
2. Ouvrir l’entrée Données démographiques.
3. Saisir comme âge minimal 30 ans, et comme âge maximal 39 ans.
4. Cliquer sur Rechercher.
5. CONFIRMEZ qu’une liste de fiches s’affiche (environ 120 Individus sur test-1).
6. Ouvrir quelques fiches.
7. Pour chaque fiche, CONFIRMEZ que l’Individu a bien entre 30 et 39 ans.
8. Cliquer sur Sélectionner Tous au-dessus de la liste.
9. Dans le menu Actions, sélectionner Etiquettes pour la Poste – imprimer.
10. Dans le menu qui s’ouvre, sélectionner un format d’étiquette.
11. Cliquer sur le bouton Imprimer des étiquettes pour la Poste.
12. CONFIRMEZ qu’un fichier PDF est crée.
13. Ouvrir le fichier PDF.
14. CONFIRMEZ que le fichier contient uniquement les adresses des trentenaires.

## 014 - Exporter dans Excel tous les habitants d’une ville donnée

1. Dans le menu Recherche, ouvrir la Recherche Avancée.
2. Ouvrir l’entrée Critères de base.
3. Dans le champ Type de contact, choisir le type Foyer.
4. Ouvrir l’entrée Champs d’adresses.
5. Dans le champ Ville, saisir le nom d’une ville.
6. Cliquer sur Rechercher.
7. CONFIRMEZ qu’une liste s’affiche.
8. Si aucune liste ne s’affiche, tester avec une autre ville (Aubure par exemple).
9. Cliquer sur Sélectionner Tous au-dessus de la liste.
10. Dans le menu Actions, sélectionner Contacts – exporter.
11. Sur la page Options d’export, choisir Exporter les champs Principaux.
12. Cliquer sur Suivant.
13. CONFIRMEZ qu’un fichier Excel est téléchargé.
14. Ouvrir le fichier Excel.
15. CONFIRMEZ que tous les Foyers de la liste sont dans le ficher Excel.

## 015 - Envoyer un mail à tous les Chefs de famille

11. Dans le menu Recherche, ouvrir la Recherche Avancée.
12. Ouvrer l’entrée Paramètres de recherche.
13. Dans le menu Type des résultats, sélectionner le menu Contacts liés.
14. Sélectionner le type de relation Chef de famille de.
15. Cliquer sur Rechercher.
16. CONFIRMEZ qu’une liste d’individus s’affiche (environ 1 050 Individus sur test-1).
17. Ouvrir quelques fiches Individus
18. Sur chaque fiche, CONFIRMEZ la présence d’une relation Chef de famille de.
19. Cliquer sur Sélectionner Tous au-dessus de la liste.
20. Dans le menu Actions, sélectionner Courriel – planifier/envoyer un mailing.
21. CONFIRMEZ que la page de création d’un mailing s’ouvre.
22. Saisir un nom pour le mailing
23. Renseigner le sujet (= l’objet qui s’affichera dans le mail)
24. Choisir Désabonnement pour le champ « Se désabonner de la liste »
25. Cliquer sur Suivant
